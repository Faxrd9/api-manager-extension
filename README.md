API 配置管理器 (SillyTavern Extension)
一个简单的 SillyTavern（酒馆）扩展工具。
它会在 API 设置区域注入一个轻量化的工具栏，支持配置的搜索、筛选、导出与备份。

A lightweight toolbar for SillyTavern's API settings page, supporting search, filtering, export, and backup of your API profiles.

🚀 功能点 (Features)
搜索增强 (Search)：输入关键词实时筛选 API 配置。

快速切换 (Quick Switch)：支持点击候选列表直接应用配置。

备份管理 (Backup)：一键导出 JSON 备份，或从备份中导入/合并。
